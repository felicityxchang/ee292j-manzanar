# data/

After Basile ingests your WACZ and photos into the class Starling instance, he will provide a spreadsheet with one row per asset. Save it here as `assets.csv`:

| Asset CID | Asset original name | group_subcollection |
|-----------|---------------------|---------------------|

The CIDs in this file are your entry points for all `starling attr` and `starling file` commands.
